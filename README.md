# TrikiClicker

![TrikiClicker Icon](icon.png)

**TrikiClicker** to zaawansowana aplikacja na system Android, stworzona jako dedykowany interfejs dla kontrolera sprzętowego "Triki". Aplikacja wykorzystuje technologię Bluetooth Low Energy (BLE) do nawiązywania bezprzewodowego połączenia z urządzeniem, umożliwiając śledzenie akcji użytkownika w czasie rzeczywistym, zliczanie kliknięć oraz zarządzanie ustawieniami urządzenia.

---

## 🚀 Kluczowe funkcje

* **Dynamiczne wykrywanie urządzeń:** Inteligentne skanowanie BLE w poszukiwaniu dostępnych kontrolerów "Triki" w otoczeniu.
* **Komunikacja w czasie rzeczywistym:** Wykorzystanie standardu Bluetooth UART do odbierania i przetwarzania pakietów danych.
* **Nowoczesny interfejs:** Aplikacja zbudowana w pełni w oparciu o **Jetpack Compose**, co zapewnia płynne animacje i nowoczesny wygląd (Material 3).
* **Personalizacja:** Obsługa trybu ciemnego i jasnego, wbudowany system resetowania wyników oraz łatwe zarządzanie połączeniem.
* **Wydajność:** Niskopoziomowe przetwarzanie strumienia danych BLE zapewnia minimalne opóźnienia w reakcji aplikacji na fizyczne kliknięcia.

## 🛠️ Technologie

* **Język:** Kotlin
* **UI:** Jetpack Compose (Material Design 3)
* **Komunikacja:** Bluetooth Low Energy (BLE) API
* **Architektura:** MVVM (Model-View-ViewModel)

## 📱 Wymagania

* Android 8.0 (API 26) lub nowszy.
* Włączony moduł Bluetooth oraz uprawnienia do lokalizacji (wymagane przez API Androida dla skanowania BLE).
